# VGN Ticketpreisvergleich

Dieses Projekt vergleicht die Kosten verschiedener Ticketoptionen im VGN-Gebiet und visualisiert die Ergebnisse in einem Diagramm.

## Verglichene Ticketarten

- Deutschlandticket
- Egon
- Einzelfahrten Online
- Tagesticket

## Funktionen

Der Nutzer kann folgende Werte eingeben:

- Anzahl der Fahrten pro Tag
- Tarifstufe
- Streckenlänge
- EGON-Sonderfall

Anschließend werden die kumulierten Kosten für einen Zeitraum von einem Monat berechnet und als Liniendiagramm dargestellt.
